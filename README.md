# ROAD_ACCIDENT_DASHBOARD_USING_EXCEL

🚦 Road Accident Data Analysis Dashboard

📊 Project Overview

This project analyses road accident data and presents insights using a visual dashboard built in Microsoft Excel.

The goal of the project is to explore accident patterns and understand how different factors such as vehicle type, road conditions, location, and lighting conditions influence accident cases.

The dashboard transforms raw accident data into visual insights that are easy to understand and interpret.

⸻

🎯 Project Goal

The purpose of this project is to demonstrate how data analysis can be used to understand road accident trends and highlight factors that may contribute to accidents.

Through visualisation and analysis, the dashboard helps identify patterns in accident severity, vehicle involvement, environmental conditions, and accident locations.

⸻

📂 Dataset Description

The dataset used in this project contains information related to road accident casualties.

Each record includes details describing the accident conditions, such as:
	•	Accident severity
	•	Vehicle type involved
	•	Road surface condition
	•	Road type
	•	Lighting conditions during the accident
	•	Location of the accident
	•	Date of occurrence

These variables allow the analysis of accident behaviour under different circumstances.

⸻

🛠 Tools and Technologies

The following tools were used in this project:
	•	Microsoft Excel
	•	Pivot Tables
	•	Data Cleaning
	•	Data Analysis
	•	Data Visualisation
	•	Dashboard Design

Excel was used to organise the dataset, perform analysis, and design the final dashboard.

⸻

📊 Dashboard Features

The dashboard contains multiple analytical views that help understand accident patterns.

Key features include:
	•	Accident casualty overview
	•	Accident severity analysis
	•	Monthly accident trend analysis
	•	Vehicle type analysis
	•	Road surface condition analysis
	•	Urban and rural accident comparison
	•	Lighting condition analysis
	•	Road type analysis

Interactive filters allow users to explore the dashboard dynamically.

⸻

💡 Key Insights

The analysis highlights several important observations:
	•	Passenger vehicles are frequently involved in accident cases.
	•	Urban areas tend to report more accidents than rural areas.
	•	Many accidents occur during daylight conditions.
	•	Road conditions can influence accident patterns.
	•	Certain road types appear to experience accidents more often.

These insights demonstrate how accident data can reveal patterns that may support better road safety awareness.
